search(query, count=50, order='date', channel_id=None, event_type=None, location=None, location_radius=None, published_after=None, published_before=None, region_code=None, relevance_language=None, safe_search=None, topic_id=None, video_caption=None, video_category_id=None, video_definition=None, video_dimension=None, video_duration=None, video_embeddable=None, video_license=None, video_syndicated=None, video_type=None, **kwargs)¶

Search

Parameters:
  • query – The search query
  • count – The number of results to return
  • order – The order of the results. Can be ‘date’, ‘rating’, ‘relevance’, ‘reviewCount’, ‘title’
  • channel_id – The id of the channel (not its name). Restricts results to only those from the channel
  • event_type – The broadcast event type. Can be ‘completed’, ‘live’, ‘upcoming’
  • location – The location’s coordinates. In lat,lng format
  • location_radius – The radius the location can be in. Can use m or km
  • published_after – RFC 3339 formatted time
  • published_before – RFC 3339 formatted time
  • region_code – two-letter ISO country code
  • relevance_language – Search results most relevant to ISO 639-1 two-letter language code
  • safe_search – Can be ‘moderate’, ‘none’, ‘strict’
  • topic_id – Freebase topic id
  • video_caption – Videos containing captions. Can be ‘any’, ‘closedCaption’, ‘none’
  • video_category_id – The youtube video category id
  • video_definition – The resolution of the video. Can be ‘any’, ‘high’, ‘standard’
  • video_dimension – The number of dimensions. Can be ‘any’, ‘2d’, ‘3d’
  • video_duration – The duration of the video. Can be ‘any’, ‘long’, ‘medium’, ‘short’ (x>0, x>20, 20>x>4, 4>x)
  • video_embeddable – Can the video be embedded
  • video_license – The video’s license. Can be ‘any’, ‘creativeCommon’, ‘youtube’
  • video_syndicated – Can the video be played outside youtube.com
  • video_type – The type of video. Can be ‘any’, ‘episode’, ‘movie’
Returns:

The list of results
